# Структура Базы Данных

## Таблицы

### Photos
| Поле           | Тип            | Описание                     |
|----------------|----------------|------------------------------|
| `photo_id`     | SERIAL PK      | Уникальный ID фото           |
| `original_url` | TEXT NOT NULL  | URL к оригинальному файлу    |
| `feed_url`     | TEXT           | URL к уменьшенному для ленты |
| `thumb_url`    | TEXT           | URL к миниатюре              |

---

### Accounts
| Поле              | Тип             | Описание               |
|-------------------|-----------------|------------------------|
| `id`              | SERIAL PK       | Уникальный ID аккаунта |
| `username`        | VARCHAR(50) UK  | Имя пользователя       |
| `avatar_photo_id` | INT FK          | ID фото аватара        |
| `bio`             | TEXT            | Описание профиля       |
| `created_at`      | TIMESTAMP       | Дата создания          |

---

### Posts
| Поле            | Тип          | Описание                       |
|-----------------|--------------|--------------------------------|
| `id`            | SERIAL PK    | Уникальный ID поста            |
| `account_id`    | INT FK       | ID автора поста                |
| `title`         | VARCHAR(255) | Заголовок поста                |
| `headerContent` | TEXT         | Краткое содержание (для ленты) |
| `fullContent`   | TEXT         | Полное содержание              |
| `created_at`    | TIMESTAMP    | Дата создания                  |
| `updated_at`    | TIMESTAMP    | Дата обновления                |

---

### PostPhotos
| Поле          | Тип                   | Описание               |
|---------------|-----------------------|------------------------|
| `post_id`     | INT FK                | ID поста               |
| `photo_id`    | INT FK                | ID фото                |
| `sort_order`  | INT                   | Порядок фото в посте   |
| **PK**        | `(post_id, photo_id)` | Уникальность связи     |

---

### Comments
| Поле         | Тип         | Описание                           |
|--------------|-------------|------------------------------------|
| `id`         | SERIAL PK   | Уникальный ID комментария          |
| `account_id` | INT FK      | ID автора комментария              |
| `post_id`    | INT FK      | ID поста, к которому комментарий   |
| `content`    | TEXT        | Текст комментария                  |
| `created_at` | TIMESTAMP   | Дата создания                      |
| `updated_at` | TIMESTAMP   | Дата обновления                    |

---

### Likes
| Поле         | Тип                     | Описание                          |
|--------------|-------------------------|-----------------------------------|
| `account_id` | INT FK                  | ID пользователя, который лайкнул  |
| `post_id`    | INT FK                  | ID поста, который лайкнули        |
| `created_at` | TIMESTAMP               | Дата лайка                        |
| **UK**       | `(account_id, post_id)` | Уникальность лайка                |

---

### Views
| Поле         | Тип                     | Описание                           |
|--------------|-------------------------|------------------------------------|
| `account_id` | INT FK                  | ID пользователя, который посмотрел |
| `post_id`    | INT FK                  | ID поста, который посмотрели       |
| `created_at` | TIMESTAMP               | Дата просмотра                     |
| **UK**       | `(account_id, post_id)` | Уникальность просмотра             |

---

### Shares
| Поле         | Тип                     | Описание                           |
|--------------|-------------------------|------------------------------------|
| `account_id` | INT FK                  | ID пользователя, который поделился |
| `post_id`    | INT FK                  | ID поста, которым поделились       |
| `created_at` | TIMESTAMP               | Дата публикации                    |
| **UK**       | `(account_id, post_id)` | Уникальность публикации            |

---

### AccountComplaints
| Поле                 | Тип        | Описание                              |
|----------------------|------------|---------------------------------------|
| `id`                 | SERIAL PK  | Уникальный ID жалобы                  |
| `sender_account_id`  | INT FK     | ID пользователя, который жалуется     |
| `account_id`         | INT FK     | ID пользователя, на которого жалуются |
| `cause`              | TEXT       | Причина жалобы                        |
| `created_at`         | TIMESTAMP  | Дата подачи жалобы                    |

---

### PostComplaints
| Поле                 | Тип         | Описание                           |
|----------------------|-------------|------------------------------------|
| `id`                 | SERIAL PK   | Уникальный ID жалобы               |
| `sender_account_id`  | INT FK      | ID пользователя, который жалуется  |
| `post_id`            | INT FK      | ID поста, на который жалуются      |
| `cause`              | TEXT        | Причина жалобы                     |
| `created_at`         | TIMESTAMP   | Дата подачи жалобы                 |

---

### CommentComplaints
| Поле                 | Тип         | Описание                             |
|----------------------|-------------|--------------------------------------|
| `id`                 | SERIAL PK   | Уникальный ID жалобы                 |
| `sender_account_id`  | INT FK      | ID пользователя, который жалуется    |
| `comment_id`         | INT FK      | ID комментария, на который жалуются  |
| `cause`              | TEXT        | Причина жалобы                       |
| `created_at`         | TIMESTAMP   | Дата подачи жалобы                   |
