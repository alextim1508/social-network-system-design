# social-network-system-design

System Design социальной сети для путешественников

## 🧭 Краткое описание

Проект представляет собой социальную сеть для путешественников, позволяющую публиковать посты с фотографиями, описаниями и геолокацией, оставлять комментарии и оценки, подписываться на других путешественников и находить популярные места для поездок.

Целевая аудитория — пользователи из стран СНГ. Планируемый рост: 10 миллионов активных пользователей в день (DAU) в первый год.

Система ориентирована на использование с мобильных устройств и браузеров, обеспечивая удобное взаимодействие и высокую надежность.

## ⚙ Основные характеристики

- **DAU**: 10 000 000 пользователей
- **Запросов на пользователя в день**: 15-20
- **Доступность**: 99.9%
- **Время отклика API**: <= 500 мс
- **Хранение данных**: постоянное
- **Регион аудитории**: страны СНГ

## 🛠️ Основные функции

- Профили пользователей
- Посты (текст, фото, геолокация, теги)
- Комментарии и лайки
- Подписки на других пользователей
- Лента постов (на основе подписок)
- Поиск (по пользователям, тегам, геолокации)
- Уведомления

## 📊 Основные метрики нагрузки

- **Итоговые RPS (чтение)**: ~4 050
- **Итоговые RPS (запись)**: ~544
- **Входящий трафик**: ~93.1 МБ/с
- **Исходящий трафик**: ~3.1 ГБ/с
- **Объем хранения данных (в месяц)**: ~241 ТБ
- **Количество дисков для хранения данных (в год)**: HDD - 230 шт, SSD SATA - 49 шт, SSD NVMe - 103 шт
